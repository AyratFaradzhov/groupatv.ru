# Руководство по категориям и оформлению товаров в products.json

Это руководство описывает правила оформления товаров в файле `data/products.json` для каждой категории, включая допустимые формы товаров и вкусы.

## Структура товара

Каждый товар в `products.json` должен иметь следующую структуру:

```json
{
  "id": "уникальный-id",
  "name": "Название товара",
  "nameRu": "Название товара на русском",
  "nameEn": "Название товара на английском",
  "brand": "бренд",
  "category": "категория",
  "image": "путь/к/изображению.webp",
  "sku": "артикул",
  "type": "тип_товара",
  "flavor": "вкус",
  "weight": "вес",
  "flags": {
    "missing_text": false,
    "missing_images": false
  },
  "sourcePath": "путь/к/исходным/файлам",
  "nutrition": {
    "kcal": null,
    "protein": null,
    "fat": null,
    "carb": null
  }
}
```

## Категории товаров

### 1. Бисквитное пирожное (`sponge cake`)

**Описание:** Товары из бисквита, включая пирожные, пончики, батончики и палочки.

**Допустимые формы товаров:**
- `biscuits` - бисквитные палочки
- `donuts` - пончики (бисквитные)
- `bars` - батончики (бисквитные)
- `sticks` - палочки (бисквитные)
- `cupcake` - кекс

**Примеры товаров:**
- Бисквит «Дубайское пирожное»
- Бисквитный пончик
- Бисквитный батончик
- Бисквитные палочки
- Кекс «Red Love» в форме сердца

**Допустимые вкусы:**
- `sweet` - сладкий

**Пример:**
```json
{
  "id": "bs0001",
  "name": "Бисквит «Дубайское пирожное» в шоколаде",
  "category": "sponge cake",
  "type": "biscuits",
  "flavor": "sweet"
}
```

### 2. Карамель (`caramel`)

**Описание:** Карамельные конфеты, леденцы и лукум.

**Допустимые формы товаров:**
- `caramel` - карамель
- `lollipops` - леденцы
- `lokum` - лукум (турецкая сладость)

**Примеры товаров:**
- Карамель Damla New2
- Молочная карамель
- Леденцы Coffee Intense
- Леденцы Choco Mint
- Леденцы Fruty Duos
- Кислые лукум-конфеты «Мискет»

**Допустимые вкусы:**
- `sweet` - сладкий
- `sour` - кислый
- `bitter` - горький (для кофейных леденцов)

**Пример:**
```json
{
  "id": "5571",
  "name": "Карамель Damla New2 — тропик и арбуз с начинкой",
  "category": "caramel",
  "type": "caramel",
  "flavor": "sweet"
}
```

```json
{
  "id": "7341",
  "name": "Леденцы Coffee Intense со вкусом кофе и начинкой",
  "category": "caramel",
  "type": "caramel",
  "flavor": "bitter"
}
```

### 3. Конфеты (`candy`)

**Описание:** Различные виды конфет, включая сбивные, драже и шоколадные конфеты.

**Допустимые формы товаров:**
- `candies` - жевательные конфеты
- `marshmallow` - маршмеллоу (сбивные конфеты)
- `dragee` - драже
- `chocolate_candies` - шоколадные конфеты

**Примеры товаров:**
- Сбивные конфеты MELLO
- Драже в молочном шоколаде
- Драже в белом шоколаде
- Драже «Мискет» с арахисом
- Жевательные конфеты Damla
- Шоколадные конфеты

**Допустимые вкусы:**
- `sweet` - сладкий
- `sour` - кислый
- `bitter` - горький

**Пример:**
```json
{
  "id": "kf0013",
  "name": "Сбивные конфеты MELLO без глазури",
  "category": "candy",
  "type": "candies",
  "flavor": "sweet"
}
```

```json
{
  "id": "pl1515",
  "name": "Драже в молочном шоколаде со вкусом ежевики",
  "category": "candy",
  "type": "chocolate",
  "flavor": "sweet"
}
```

### 4. Мармелад (`marmalade`)

**Описание:** Жевательный мармелад различных форм.

**Допустимые формы товаров:**
- `belts` - ленты
- `pencils` - карандаши
- `bears` - мишки
- `figurative` - фигурные
- `marmalade` - мармелад (общее)

**Допустимые вкусы:**
- `sweet` - сладкий
- `sour` - кислый

**Пример:**

### 8. Желе (`jelly`)

**Описание:** Желейные десерты различных форм и упаковок.

**Допустимые формы товаров:**
- `jelly-desserts` - желейные десерты (общее название для стандартных форм)
- `jelly_pudding` - желе-пудинг (более плотная консистенция)
- `drinkable_jelly` - питьевое желе (жидкая форма в упаковке)
- `jelly_sticks` - желе в стиках/палочках (удобная порционная форма)
- `jelly_cups` - желе в стаканчиках/чашках (порционное в индивидуальной упаковке)
- `jelly_cans` - желе в банках (большая упаковка, например, банка-машинка)
- `jelly_pieces` - желе кусочками (отдельные кусочки в упаковке)
- `jelly_figures` - фигурное желе (желе в виде фигурок)

**Примеры товаров:**
- Питьевое желе Navroz Frutti
- NAVROZ Желейный десерт
- Желе FRUTTI в стиках
- Желе Frutti Car в банке-машинке

## Правила классификации

### Бисквитное пирожное
- ✅ Все товары с "бисквит", "бисквитный", "бисквитное" в названии
- ✅ Пончики из бисквита
- ✅ Батончики из бисквита
- ✅ Палочки из бисквита

### Карамель
- ✅ Все товары с "карамель" в названии
- ✅ Все леденцы (`lollipops`)
- ✅ Все товары с "лукум" в названии
- ✅ Товары с `type: "caramel"`

### Конфеты
- ✅ Сбивные конфеты (маршмеллоу-типа)
- ✅ Драже (все виды)
- ✅ Шоколадные конфеты (не плитки шоколада)
- ✅ Жевательные конфеты

### Желе
- ✅ Все товары с "желе", "желейн" в названии
- ✅ Питьевое желе (`drinkable_jelly`)
- ✅ Желе в стиках/палочках (`jelly_sticks`)
- ✅ Желе в банках (`jelly_cans`)
- ✅ Желе в стаканчиках (`jelly_cups`)
- ✅ Желейные десерты (`jelly-desserts`)
- ✅ Желе-пудинг (`jelly_pudding`)
- ✅ Фигурное желе (`jelly_figures`)

### Важные замечания

1. **Шоколадные конфеты vs Шоколад:**
   - Шоколадные **конфеты** → категория `candy`
   - Шоколадные **плитки** → категория `chocolate`

2. **Драже:**
   - Все драже относятся к категории `candy`, независимо от того, в каком шоколаде они покрыты

3. **Леденцы:**
   - Все леденцы относятся к категории `caramel`

4. **Лукум:**
   - Лукум относится к категории `caramel`

5. **Желе:**
   - Все желейные десерты относятся к категории `jelly`
   - Питьевое желе использует тип `drinkable_jelly`
   - Желе в стиках/палочках использует тип `jelly_sticks`
   - Желе в банках использует тип `jelly_cans`
   - Стандартные желейные десерты используют тип `jelly-desserts`
   - Желе-пудинг использует тип `jelly_pudding`

## Проверка корректности

Перед добавлением товара убедитесь:
1. ✅ Категория соответствует типу товара
2. ✅ Тип товара (`type`) соответствует категории
3. ✅ Вкус (`flavor`) допустим для данной категории
4. ✅ Все обязательные поля заполнены
5. ✅ Пути к изображениям корректны

## Частые ошибки

❌ **Неправильно:**
```json
{
  "name": "Драже в молочном шоколаде",
  "category": "chocolate"  // ❌ Драже должно быть в candy
}
```

✅ **Правильно:**
```json
{
  "name": "Драже в молочном шоколаде",
  "category": "candy"  // ✅
}
```

❌ **Неправильно:**
```json
{
  "name": "Леденцы Coffee Intense",
  "category": "candy"  // ❌ Леденцы должны быть в caramel
}
```

✅ **Правильно:**
```json
{
  "name": "Леденцы Coffee Intense",
  "category": "caramel"  // ✅
}
```

## Справочник форм товаров

Подробное описание всех форм товаров с человекочитаемыми названиями:

| Код формы | Название | Описание |
