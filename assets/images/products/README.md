# Каталог для hero-каруселей на product.html

Положите сюда файлы (обязательно имена в **нижнем регистре** для Linux-сервера):

- `product_cover_one.webp`
- `product_cover_two.webp`
- `product_cover_three.webp`
- `product_cover_four.webp`
- `brand_cover_one.webp`
- `brand_cover_two.webp`

Без этих файлов на сервере будут 404. После добавления задеплойте папку `assets/images/products/`.
